# Student Task Manager

This is a small web app I built to practice backend development using Flask.

## Features
- Add and remove tasks
- Simple interface
- Tasks update instantly on the page

## Tech Stack
- Python
- Flask
- Basic HTML

## How it works
You can type a task, add it to the list, and delete it when you're done.  
Everything updates immediately without refreshing the page.

## Why I built this
I wanted to get hands-on experience with building a simple web app and understand how routing and user input work in Flask.

## Run locally
```bash
pip install flask
python app.py
